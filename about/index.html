This website is powered by Jekyll and hosted on GitHub. It supports different screen sizes. You can view this website through your desktop, laptop or mobile devices, like pad or smart phones. All of the pages within this website are static. The postings of portfolio, tutorial and blog are all created with Markdown. Take a look at Markdown Cheatsheet from here.

Here are some resources which helped me to build this website:

  • Two blogs introduce how to setup website on Github, Creating and Hosting a Personal Site on GitHub and How I Created a Beautiful and Minimal Blog.
  • The official Jekyll website, along with detailed documentation.
  • Documentation on GitHub about hosting static webpages with GitHub pages.
  • Tutorial for how to setup personal website with Jekyll locally.
  • The source code repository on GitHub for this website.
  • The list page template for Portfolio.
  • The personal page template for CV.

If you have any questions or comments, please post them below. If you like my website, you can share it with your followers or follow me on Twitter!
